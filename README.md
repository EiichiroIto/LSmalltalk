# LSmalltalk
Little Smalltalk Implementation by Pharo Smalltalk.

It can
- read imageSource of Lst4
- generage image
- write image file compatible with Lst4
- execute bytecodes of Lst4

# Examples
see LImageBuilder class examples.

```smalltalk
LImageBuilder example6.
```
